## 🚀 核心功能全解析

### 1. 毫秒级高精度追踪
*   **高德 3D 引擎**：基于高德地图 SDK，实现室内外无缝定位。
*   **实时详细地址**：通过逆地理编码，将经纬度实时转化为直观的文字地址（如“XX 路 XX 弄 XX 号”）。
*   **动态距离监测**：实时计算孩子当前位置与“安全中心”的直线距离。

### 2. 电子围栏 & 自动报警
*   **智能地图选点**：在解锁状态下，可直接点击地图任意位置更换“安全中心”，无需繁琐清除。
*   **自动化响应**：一旦孩子离开设定的安全半径（如 500 米），系统将在后台自动触发：
    *   **自动拨号**：第一时间拨打预设的家长电话。
    *   **自动短信**：发送包含孩子精确地址的提醒短信。

### 3. 三重安全防御体系
*   **全局配置锁**：任何关键配置修改均需密码确认，防止误操作或恶意修改。
*   **防卸载守护 (Device Admin)**：利用系统底层权限，使应用在常规应用列表中无法被卸载。
*   **后台弹出界面**：针对国产 ROM 优化，确保在锁屏或后台状态下仍能成功发起预警。

### 4. 增强型后台运行
*   **前台服务保障**：通过常驻通知栏，确保应用在锁屏或系统内存紧张时依然保持运行。
*   **可视化状态监控**：首页卡片实时显示“正在实时守护中”或“守护已停止”，一目了然。

### 5. [新功能] 告警测试与验证
*   **一键模拟预演**：新增“测试报警”按钮，允许家长在不离开安全区的情况下，模拟触发电话与短信告警。
*   **通道可靠性验证**：实时验证家长手机是否能正常接收报警短信，确保紧急时刻通道畅通。

## 🛠 快速上手指南
1.  **初次启动**：请授予“始终允许”定位、拨号、短信及通知权限。
2.  **激活底层保护**：
    *   点击“立即开启防卸载保护”红色按钮并激活。
    *   点击右上角铃铛，在系统设置中开启“儿童安全守护检测插件”辅助功能。
3.  **解锁并配置**：
    *   点击右上角**红色锁头**，输入管理密码 `123456`。
    *   在地图上直接点击孩子常在的位置（如学校、家）作为安全中心。
    *   设置安全半径（米）和接收报警的手机号。
4.  **测试与开启**：
    *   点击“测试报警”按钮，确认您的手机能收到模拟短信或接到电话。
    *   点击“保存并开启守护”，应用将进入后台监控模式。
    *   如需修改，需再次点击锁头按钮进行解锁。
