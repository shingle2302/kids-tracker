## 🚀 核心功能全解析

### 1. 毫秒级高精度追踪
*   **高德 3D 引擎**：基于高德地图 SDK，实现室内外无缝定位。
*   **实时详细地址**：通过逆地理编码，将经纬度实时转化为直观的文字地址。
*   **动态距离监测**：实时计算孩子当前位置与“安全中心”的直线距离。

### 2. 电子围栏 & 自动报警
*   **多维响应机制**：一旦离开安全半径，系统自动触发：
    *   **自动拨号**：第一时间拨打家长电话。
    *   **自动短信**：发送包含孩子精确地址的提醒短信。
    *   **[新增] 自动录音**：越界瞬间自动开启环境录音，为安全提供关键声音凭证。

### 3. [升级] 管理员权限与安全闭环
*   **密码自定义**：支持修改默认管理员密码，将权限掌握在家长手中。
*   **首次强制引导**：首次使用默认密码解锁后，系统强制引导用户设置新密码，彻底告别“弱口令”隐患。
*   **全局配置锁**：任何关键配置修改均需密码确认，且修改后自动进入锁定状态。

### 4. 增强型后台运行
*   **Android 14+ 深度适配**：完美兼容最新系统的前台服务要求，支持后台持续定位与麦克风访问。
*   **测试模式隔离**：新增专门的测试指令，允许在不开启真实守护的情况下验证短信、电话及录音功能。

## 🛠 快速上手指南
1.  **初次启动**：请授予“始终允许”定位、拨号、短信、录音及通知权限。
2.  **激活底层保护**：
    *   点击“立即开启防卸载保护”红色按钮并激活。
    *   在辅助功能中开启“儿童安全守护检测插件”。
3.  **首次解锁与密码设置**：
    *   点击右上角**红色锁头**，输入默认密码 `123456`。
    *   **重要**：首次解锁后请根据系统提示设置您的私人密码。设置成功后，界面将自动锁定。
4.  **配置与测试**：
    *   再次解锁后，在地图上点击选择安全中心点。
    *   设置安全半径和接收报警的手机号。
    *   勾选需要的告警方式，点击“测试已选告警方式”进行模拟验证。
5.  **正式开启**：
    *   点击“保存并开启守护”，应用将进入后台全天候监控模式。
